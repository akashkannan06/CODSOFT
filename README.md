# Titanic Survival Prediction

## Objective
Predict whether a passenger survived the Titanic disaster using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Algorithm Used
Random Forest Classifier

## Features
- Passenger Class
- Gender
- Age
- Fare
- Family Details
- Embarked Port

## Result
The model achieved good prediction accuracy.
